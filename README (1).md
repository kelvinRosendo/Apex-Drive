# 🏎️ APEX DRIVE

Projeto desenvolvido para o trabalho de **Desenvolvimento de Sistemas** da **ETEC**, simulando o site de uma empresa fictícia de locação de veículos de luxo.

---

## 🖥️ Demonstração

> Abra o arquivo `index.html` no navegador para visualizar o projeto.

---

## 📌 Sobre

A **APEX DRIVE** é uma landing page estática de uma locadora de supercarros. O objetivo do projeto foi praticar **HTML, CSS e JavaScript puro**, explorando animações, interatividade e design responsivo sem uso de frameworks.

---

## ⚙️ Funcionalidades

- Cursor personalizado com efeito de inércia
- Navbar que muda de estilo ao rolar a página
- Carrossel de veículos com auto-avanço, dots e suporte a swipe (mobile)
- Animações de entrada com `IntersectionObserver`
- Contadores animados nas estatísticas do hero
- Efeito parallax no background
- Formulário de reserva com validação e feedback de sucesso
- Marquee infinito com as marcas da frota
